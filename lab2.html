Лабораторная работа №2: Линейный вычислительный процесс

Цели, задачи, требования

Цели и задачи работы: изучение функций ввода-вывода данных, программирование вычисления значения выражения.

Задание к работе: реализовать линейный вычислительный процесс. Самостоятельно решить задачу в соответствии с индивидуальным вариантом.

Требования к отчету: отчет должен содержать титульный лист, задание, блок-схему алгоритма решения задачи, текст программы с комментариями, тесты.

Задание 1

Дана сторона квадрата a. Найти его площадь: S = a².

Код выполнения:

function calculateSquare() {
    let a = prompt("Введите сторону квадрата:");
    if (a !== null && !isNaN(a)) {
        let S = a * a;
        alert("Площадь квадрата: " + S);
    } else {
        alert("Пожалуйста, введите числовое значение.");
    }
}

Задание 2

Дана масса M в килограммах. Используя операцию деления нацело, найти количество полных тонн в ней (1 тонна = 1000 кг).

Код выполнения:

function calculateTons() {
    let M = prompt("Введите массу в килограммах:");
    if (M !== null && !isNaN(M)) {
        let tons = Math.floor(M / 1000);
        alert("Количество полных тонн: " + tons);
    } else {
        alert("Пожалуйста, введите числовое значение.");
    }
}
